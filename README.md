প্রজেক্ট ডকুমেন্টেশন: চাহিদা-যোগান (ই-কমার্স)এই প্রজেক্টটি একটি Single Page Application (SPA) যা HTML, Tailwind CSS এবং Firebase ব্যবহার করে তৈরি করা হয়েছে। এটিতে ডায়নামিক প্রোডাক্ট লোডিং, কার্ট সিস্টেম, ইউজার অথেন্টিকেশন এবং একটি ফিচার-সমৃদ্ধ অ্যাডমিন প্যানেল রয়েছে।১. প্রযুক্তি এবং সেটআপ (Tech Stack)ফ্রন্টএন্ড: HTML5, JavaScript (ES6 Modules)ডিজাইন ফ্রেমওয়ার্ক: Tailwind CSS (CDN)আইকন: FontAwesome (CDN)ফন্ট: Google Fonts (Hind Siliguri)ব্যাকএন্ড ও ডাটাবেস: Firebase Authentication & Firestore Database (v11.6.1)ইমেইল সার্ভিস: Formspree (ফুটারের কন্টাক্ট ফর্মের জন্য)২. ইউজার ইন্টারফেস (UI) ও ফিচারসমূহহিরো সেকশন (Hero Section)ডায়নামিক স্লাইডার: অ্যাডমিন প্যানেল থেকে নিয়ন্ত্রিত অটোমেটিক স্লাইডার।কন্টেন্ট: প্রতিটি স্লাইডে টাইটেল, সাব-টাইটেল এবং "অর্ডার করুন" বাটন।ফলব্যাক: ডাটাবেসে ব্যানার না থাকলে ডিফল্ট ৫টি ব্যানার দেখাবে।নেভিগেশন (Navigation)ডেস্কটপ: টপ বারে ক্যাটাগরি মেনু।মোবাইল: হ্যামবার্গার মেনু সহ সাইড ড্রয়ার (Side Drawer)।স্টিকি হেডার: স্ক্রল করলেও মেনু উপরে ফিক্সড থাকবে।প্রোডাক্ট ডিসপ্লে ও ফ্রি বই ফিচারক্যাটাগরি ভিত্তিক: প্রতিটি ক্যাটাগরি আলাদা সেকশনে দেখাবে।ফ্রি বই লজিক: বইয়ের দাম ০ টাকা হলে:কভারে লাল রঙের "ফ্রি" স্টিকার দেখাবে।"কার্টে নিন" বাটনের পরিবর্তে "ডাউনলোড করুন" বাটন আসবে।লগইন ছাড়াই সরাসরি পিডিএফ ডাউনলোড করা যাবে।পেইড বই: সাধারণ বইয়ের ক্ষেত্রে "কার্টে নিন" বাটন এবং দাম দেখাবে।মডেল ভিউ: পণ্যের বিস্তারিত, লেখকের নাম এবং বড় ছবি দেখার জন্য পপ-আপ মডেল।শপিং কার্ট ও চেকআউট (Cart & Checkout)সাইডবার কার্ট: ডান পাশ থেকে স্লাইড হয়ে আসা কার্ট।শিপিং চার্জ: ঢাকার ভেতরে (২০ টাকা) এবং বাহিরে (৬০ টাকা) রেডিও বাটন সিলেকশন।পেমেন্ট মেথড: * ক্যাশ অন ডেলিভারি (COD)।বিকাশ (সিলেক্ট করলে পার্সোনাল নাম্বার এবং TrxID ইনপুট ফিল্ড দেখাবে)।ইউজার ড্যাশবোর্ড (User Dashboard)লগইন থাকা অবস্থায় ইউজার নিজের সব অর্ডার দেখতে পারবে।অর্ডারের স্ট্যাটাস (Pending, Delivered, Cancelled) কালার কোড সহ দেখা যাবে।৩. অ্যাডমিন প্যানেল (Admin Panel)অ্যাক্সেস: শুধুমাত্র নির্দিষ্ট ইমেইল (eco452@gmail.com) দিয়ে লগইন করলে "Admin" বাটন দৃশ্যমান হবে।ট্যাব সিস্টেম: ৩টি প্রধান ট্যাব -
